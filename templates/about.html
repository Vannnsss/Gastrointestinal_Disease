Model yang digunakan

  • Website ini menggunakan dua model deep learning utama, yaitu ConvMixer dan ResNet, untuk melakukan klasifikasi dan deteksi penyakit gastrointestinal berbasis citra digital.

Teknologi yang digunakan

  • Frontend:
    -HTML5, CSS3, dan JavaScript.
    -Framework Bootstrap dan FullPage.js untuk desain interaktif dan navigasi.
    Backend:
    -Flask sebagai framework backend untuk menangani request dan routing.
    -Python untuk pengolahan data dan pengelolaan model machine learning.
    Model Deployment:
    -Framework TensorFlow dan Keras untuk integrasi model machine learning ke dalam sistem.

Jenis Penyakit yang dideteksi

  • Sistem ini mampu mendeteksi beberapa jenis penyakit gastrointestinal seperti:
    -Esophagitis.
    -Ulcerative Colitis.
    -Polyps.
    -Dan penyakit gastrointestinal lainnya sesuai dengan dataset Kvasir.
    Fitur Website
    Prediksi Penyakit:
    Pengguna dapat mengunggah gambar medis (format JPG) untuk dianalisis.
    Model ConvMixer akan memberikan hasil diagnosis secara cepat.
